Catalogue - downloaded SMCE and Better Regulation library

This page tracks the directly downloaded PDF library currently stored in raw/ for the SMCE wiki.

It is narrower than Master bibliography and Catalogue - official EU and JRC sources. Its purpose is operational: these are the sources that are already downloaded locally, processed into extracted text, and ready for direct reuse in note-writing.

What is in scope

  • sources downloaded into raw/
  • extracted text cached in raw/extracted-text/
  • provenance manifests stored in raw/source-manifests/
  • especially useful sources for SMCE, SOCRATES, Better Regulation Guidelines and Toolbox, EC impact assessment, and Sensitivity and robustness analysis

Downloaded library

Title Year Type Local file Main use in the wiki Priority
SOCRATES Manual 2022 JRC manual raw/socrates-manual-2022.pdf Main operational reference for the SOCRATES workflow, including multi-criteria, equity, and sensitivity modules High
A Social Multi-Criteria Framework for Ex-Ante Impact Assessment: Operational Issues 2017 JRC technical report raw/smce-operational-issues-2017.pdf Best concise JRC explanation of how SMCE maps onto ex ante impact assessment structure High
On the Use of Cost-Benefit Analysis and Multi-Criteria Evaluation in Ex-Ante Impact Assessment 2017 JRC technical report raw/cba-vs-mce-jrc-2017.pdf Strong bridge text for explaining why SMCE/MCE may be preferable to pure CBA in plural-value policy contexts High
20 years of Social Multi-Criteria Evaluation in policy assessment 2025 JRC science-for-policy report raw/smce-20-year-review-2025.pdf Best overview of applications, process completeness, strengths, and recurring implementation problems High
Better Regulation Toolbox 2025 European Commission toolbox raw/better-regulation-toolbox-2023.pdf Main practical reference for official impact-assessment workflow, tools, and reporting expectations High
Uncertainty and Sensitivity Analysis for policy decision making 2020 JRC technical report raw/uncertainty-sensitivity-jrc-2020.pdf Best downloaded methodological support for local/global sensitivity analysis and uncertainty framing High

Recommended reading order from the downloaded set

  1. Better Regulation Guidelines and Toolbox
  2. SOCRATES
  3. SMCE
  4. Sensitivity and robustness analysis

In terms of the actual local files, a good sequence is: 1. raw/better-regulation-toolbox-2023.pdf 2. raw/socrates-manual-2022.pdf 3. raw/smce-operational-issues-2017.pdf 4. raw/cba-vs-mce-jrc-2017.pdf 5. raw/smce-20-year-review-2025.pdf 6. raw/uncertainty-sensitivity-jrc-2020.pdf

How this library enriches the wiki

  • It upgrades the wiki from link-only bibliography to a working local research pack.
  • It gives a reusable primary-source set for future note extraction.
  • It supports traceable updates to source-derived notes without relying on live web access.
